!!! note
    Убедительно просим коммитить, если вы уже выполнили задание, которого тут нет: [GitHub](https://github.com/mraliscoder/web-dvfu-gdz)

## Задание 1
```html
<html>
    <head>
        <meta charset="UTF-8" />
        <title>Заголовок</title>
    </head>
    <body text="navy">
        <p>Абзац - это выделенный фрагмент документа.</p>
        <p>Если нужно отделить один блок текста от другого - заключите блоки в теги p.</p>
    </body>
</html>
```
