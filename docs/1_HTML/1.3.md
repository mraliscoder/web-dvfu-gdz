## Задание 1
```html
<html>
    <head>
        <meta charset="UTF-8" />
        <title>1212</title>
    </head>
    <body text="#000080" background="https://stepik.org/media/attachments/course/114064/bg_2.jpg">
        <h2 align="center">Атрибуты тега</h2>
        <hr width="80%" color="#000080">
        <p align="justify">Тег может иметь атрибуты, дающие дополнительные возможности описания элемента html-страницы. Значение атрибутов указываются:</p>
        <ul type="square">
            <li>в двойных кавычках;</li>
            <li>в одинарных кавычках;</li>
            <li>без кавычек, если атрибут состоит из одного слова или числа.</li>
        </ul>
    </body>
</html>
```

## Задание 2
```html
<html>
    <head>
        <meta charset="UTF-8" />
        <title>Фон страницы</title>
    </head>
    <body>
        <h3 align="center">Атрибуты тега body</h3>
        <p>Атрибуты для изменения фона и цвета текста:</p>
        <ol type="I">
            <li>bgcolor - определяет цвет фона страницы;</li>
            <li>text - определяет цвет текста на странице;</li>
            <li>background - устанавливает фоновый рисунок на страницу;</li>
        </ol>
        <p>Атрибуты для изменения отступов:</p>
        <ol type="I" start="4">
            <li>topmargin - от верхнего края окна браузера до содержимого страницы;</li>
            <li>bottommargin - от нижнего края;</li>
            <li>leftmargin - от левого края;</li>
            <li>rightmargin - отступ от правого края.</li>
        </ol>
    </body>
</html>
```

    </head>

        <h2>Атрибуты тегов body</h2>
        <dl>
            <dt>body</dt>
            <dd>bgcolor - цвет фона страницы;<br>text - цвет текста на странице;<br>background - фоновый рисунок.</dd>
            <dt>ul</dt>
            <dd>type - тип маркера.</dd>
            <dt>ol</dt>
            <dd>type - тип нумерации;<br>start - номер первого элемента.</dd>
        </dl>
    </body>
