## Задание 1
```html
<html>
    <head>
        <meta charset="UTF-8" />
        <title>1212</title>
    </head>
    <body>
        Даны <b>N</b> точек на плоскости: (x<sub>1</sub>, y<sub>1</sub>), ..., (x<sub>n</sub>, y<sub>n</sub>). Найти точку (x<sup>0</sup>, y<sup>0</sup>), <mark><u>ближайшую</u></mark> к началу координат.
    </body>
</html>
```

                </ul>
            </li>
            <li>
                Изменение шрифта.
            </li>
            <li>
                Изменение размера:

                    <li><small>уменьшение</small></li>
                </ul>
            </li>
        </ol>
        <hr>
    </body>
</html>
