## Задание 8
HTML:
```html
<div class="menu">
  <p  class=active>
    <a href="#">Блочный элемент</a>
  </p>
  <p><a href="#">Строчный элемент</a></p>
</div> 
<div class="content">
  <p>
    Блочный элемент - это элемент, который занимает всю доступную ширину и всегда начинается с новой строки. 
  </p> 
  <p>
    Высота блока зависит от его содержимого. 
  </p>
</div> 
```
CSS:
```css
.menu {
    overflow: hidden;
}
.menu p {
    display: inline;
    padding: 10px;
    background: #C9A66B;
}
.menu a {
    text-decoration: none;
    color: #AF4425;
}
.menu .active {
    background: #EBDCB2;
}
.content {
    width: 500px;
    background: #EBDCB2;
    color: #662E1C;
    padding: 10px;
}

.content p {
    margin: 0.5em;
}
```
