## Задание 7
HTML:
```html
<p>Теги оформления текста:</p>  
<ul>
  <li><kbd>b</kbd> - полужирный шрифт;</li>
  <li><kbd>i</kbd> - курсивный шрифт;</li>
  <li><kbd>sup</kbd> - верхний индекс;</li>
  <li><kbd>sub</kbd> - нижний индекс.</li>  
</ul>
```
CSS:
```css 
p::first-letter{
    color: green;
    font-size: 2em;
    vertical-align: middle;
}
kbd::before{
    content: "<";
}
kbd::after{
    content: ">";
}
li::marker{
    content: "\2714";
    color: green;
}
```
