## Задание 4
HTML:
```html
<body>
  <header>
    <h2>WEB-инструменты</h2>
	<h2>HTML</h2>
	<h2>CSS</h2>
	<h2>JavaScript</h2>
  </header>  
  <nav>
    <a href="#">Основы CSS</a>
	<a href="#">Типы HTML-элементов</a>
	<a href="#">Блочные элементы</a>
	<a href="#">Строчные элементы</a>
	<a href="#">Строчно-блочные элементы</a>
  </nav>  
  <article>
	<section>
	  <h3>Типы элементов</h3>
		<p>
		  HTML5 отказался от строгого разделения тегов на блочные и строчные, но само 
          разделение, конечно, сохранилось. Только оно осуществляется через CSS. 
          Теперь у элемента есть стилевое оформление:
		</p>
		<ul>         
		   <li>блочное;</li>
		   <li>строчное;</li>
		   <li>строчно-блочное.</li>
		 </ul>    
      </section>
	  <section> 
		<h3>Универсальные стили</h3>
		<p>
		  <img src="https://stepik.org/media/attachments/course/114064/block.png">
		  Для элемента определяются:<br>
		  <b>border</b> - граница, которая может быть разной толщины, цвета, типа;<br>
		  <b>margin</b> - внешние отступы до охватывающего элемента страницы;<br>
		  <b>padding</b> - внутренние отступы от границы до содержимого блока;<br>
		  <b>background</b> - цвет фона или фоновая картинкa.
		</p>
	</section>  
  </article>
</body>
```
CSS:
```css
article {
    width: 70%;
    background: white;
    padding: 5px;
    margin: 5px 0; 
}
article img {
    float: left;
    width: 50%;
    margin: 0 5px;
}
@media (max-width: 600px){
    article {
        width: 100%;
    }
    article img {
        width: 100%;
    }
}
```
