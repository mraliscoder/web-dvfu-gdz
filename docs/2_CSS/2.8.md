## Задание 4
HTML:
```html
<div class="container">
   <div class="item">
      <b>align-items</b>
      <br>определяет выравнивание вдоль поперечной оси, расположенной перпендикулярно главной оси во flex-контейнере
  </div>
    <div class="item">
      <b>justify-content</b>
      <br> определяет выравнивание вдоль оси flex-контейнера
  </div>
    <div class="item">
      <b>flex-wrap</b>
      <br>определяет перенос вложенных элементов во flex-контейнере
  </div>
</div>
```
CSS:
```css
.container {
    display: flex;
    align-items: stretch;
    max-width: 600px;
    margin: 2px auto;
    background: LightGrey;
}
.item {
    width: 33%;
    border: solid thin grey;
    margin: 5px;
    padding: 5px;
}
@media (max-width: 600px){
    .container{
        flex-direction: column;
    }
    .item{width: inherit;}
}
```

## Задание 6
HTML:
```html
<div class="main">
    <div class="item yellow">
        желтый
    </div>
    <div class="item green">
        зеленый
    </div>
    <div class="item red">
        красный
     </div>
     <div class="item orange">
        оранжевый
    </div>
</div>
```
CSS:
```css
.main{  
    display: flex;
    height:150px;   
   }
.item{   
    padding: 5px;
    text-align:center;
    line-height:4em;
} 
.yellow{
     order: 3;
     align-self: flex-end;
     background:yellow;
}
.green{
     order: 4;
     background:green;
}    
.red{
     order: 1;
     flex-grow: 2;
     background:red;
}  
.orange{
     order: 2;
     flex-basis: 50%;
     align-self: center;
     background:orange;
} 
```
