## Задание 5
HTML:
```html
<h3>Описание стилей</h3>
<p>Описание стиля состоит из двух частей: селектора и блока определений, заключенного в фигурные скобки. Каждое определение заканчивается точкой с запятой. Определение включает свойство и значение этого свойства, разделенные двоеточием.</p>
```
CSS:
```css
h3 {
    font-style: italic;
    font-size: 0.3in;
    line-height: 3;
}
p {
    font-family: courier new;
    font-size: 12pt;
    line-height: 1.5;
}
```
